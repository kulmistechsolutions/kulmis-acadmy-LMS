# Kulmis Academy — Semantic Design System

**Source of truth for UI. Do not change color, font, or text tokens below.**

## Project Identity
- **Title:** Kulmis Academy
- **Atmosphere:** Futuristic, professional, minimal, neon-accented, fast and smooth

## Color Palette

| Role | Hex | Name |
|------|-----|------|
| Primary background | `#0B0F1A` | Deep black |
| Surface / card | `#121827` | Dark navy |
| Primary accent (buttons, links) | `#00F0FF` | Neon blue |
| Secondary accent | `#A020F0` | Neon purple |
| Tertiary accent | `#FF2D95` | Neon pink |
| Foreground / text | `#F8FAFC` | Light slate |
| Muted text | `#94A3B8` | Muted slate |
| Border | `#1E293B` | Border slate |

## Typography
- **Font family (sans):** Geist Sans — clean, modern (do not change).
- **Font family (mono):** Geist Mono — code and numbers (do not change).
- **Weights:** 400 (body), 500 (medium), 600 (semibold), 700 (bold).

## Geometry & Shape
- **Buttons / cards:** Rounded medium — `rounded-lg` (8px). Smooth, soft edges.
- **Inputs:** Same as buttons — `rounded-lg`.
- **Modals / panels:** `rounded-xl` (12px).

## Components
- **Buttons:** Glow on hover (primary), 300ms transition, subtle shadow blur.
- **Cards:** Glassmorphism — backdrop blur, border from palette.
- **Layout:** Dark mode default; neon accents only for CTAs and highlights.
